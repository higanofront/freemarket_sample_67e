# freemarket DB設計
## usersテーブル
|Column|Type|Options|
|------|----|-------|
|nickname|string|null: false|
|password|string|null: false|
|email|string|null: false|
|name|string|null: false|
|hurigana|string|null: false|
|birthday|integer|null: false|
### Association
- has_many :comments
- has_one :card
- has_one :address
- has_many :items
